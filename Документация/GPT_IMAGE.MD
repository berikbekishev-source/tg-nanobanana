# Интеграция OpenAI GPT Image 1

## Описание

GPT Image 1 — модель OpenAI для генерации изображений, поддерживающая разные размеры, качества и стили. В проекте используется для дополнительной платной модели, аналогичной NanoBanana.

## Требования

- `OPENAI_API_KEY` и, при необходимости, `OPENAI_ORGANIZATION`, `OPENAI_PROJECT_ID`.
- Доступ к API `/v1/images/generations`.

## Поддерживаемые параметры

| Параметр     | Значение по умолчанию | Допустимые значения                      |
|--------------|------------------------|-----------------------------------------|
| `size`       | `1024x1024`            | 512x512, 768x1024, 1024x768, 1024x1024, 1024x1792, 1792x1024 |
| `quality`    | `standard`             | standard, high                          |
| `background` | не используется        | transparent (опционально, только при необходимости) |

## Использование в коде

- Модель добавлена миграцией `0017_add_openai_gpt_image_model`.
- Провайдер `openai_image`.
- Генерация происходит через `botapp.services.openai_generate_images`.

## Ограничения

- `background: transparent` следует включать только при необходимости и при размерах, поддерживающих альфа-канал. По умолчанию отключено, чтобы избежать ошибок 400.
- В `response_format` используется `b64_json`, затем изображение отправляется в Telegram и Supabase.
